# Heart Disease Prediction

A Machine Learning project that uses Logistic Regression to predict the presence of heart disease based on patient medical attributes.

## Features
- Data Preprocessing
- Feature Scaling using StandardScaler
- Logistic Regression Model
- Model Evaluation using:
  - Accuracy
  - Precision
  - Recall
  - F1 Score
  - Confusion Matrix

## Technologies Used
- Python
- Pandas
- Scikit-learn
- NumPy
- Jupyter Notebook

## Results
Achieved approximately 88% accuracy on the test dataset.
